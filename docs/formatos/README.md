# Formatos de respuesta (Word)

Coloca aquí los formatos oficiales en Word para ajustar la salida jurídica.

## Dónde subir el archivo

Guarda tu archivo en esta carpeta, por ejemplo:

- `docs/formatos/Formato_Respuesta_Judicial.docx`

## Qué debe incluir el formato

Para mapearlo rápido al generador, marca secciones claras como:

1. Encabezado (despacho, radicado, partes)
2. Hechos relevantes
3. Análisis jurídico
4. Subsumción
5. Decisión
6. Firma / cierre

## Variables sugeridas

Si quieres, puedes dejar marcadores como estos dentro del texto:

- `{{radicado}}`
- `{{despacho}}`
- `{{demandante}}`
- `{{demandado}}`
- `{{decision}}`
- `{{fundamento}}`
- `{{motivacion}}`
- `{{articulos_aplicados}}`

Con ese archivo en esta ruta, en la siguiente iteración lo conecto para que el documento final siga exactamente tu formato institucional.
